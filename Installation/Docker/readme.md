# Installation des Room Booking Systems

Diese Anleitung richtet sich an Benutzer **ohne technisches Vorwissen**.

## Empfohlene Methode: Automatische Installation

### Windows
1. Öffnen Sie den Ordner `Installation`
2. Doppelklicken Sie auf `Windows-install.bat`
3. Folgen Sie den Anweisungen auf dem Bildschirm

### Linux / macOS
1. Terminal öffnen
2. In den Ordner `Installation` wechseln
3. Befehl ausführen:
   ```bash
   ./Linux-install.sh
